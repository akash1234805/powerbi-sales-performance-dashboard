# 📊 Power BI Sales Performance Dashboard

## 📌 Project Overview

This project presents a comprehensive **Sales Performance Dashboard** built using Power BI. It analyzes key business metrics such as revenue, profit, orders, and customer behavior using the AdventureWorks dataset.

The dashboard is designed to help stakeholders quickly understand sales trends, identify top-performing products, and make data-driven business decisions.

---

## 🎯 Objective

* Analyze sales performance over time (2020–2022)
* Track key KPIs like Revenue, Profit, Orders, and Return Rate
* Identify top products and categories
* Understand customer segmentation and behavior
* Compare actual performance against targets

---

## 🛠️ Tools & Technologies

* Power BI Desktop
* Power Query (Data Cleaning & Transformation)
* DAX (Data Analysis Expressions)

---

## 📁 Dataset

This project uses the **AdventureWorks dataset**, which is a publicly available dataset for learning and analysis purposes.

Dataset includes:

* Sales Data (2020, 2021, 2022)
* Customer Information
* Product Details (Categories & Subcategories)
* Returns Data
* Territory & Region Data
* Calendar Data

### 🔹 Customer Analysis

## 🔍 Key Insights

* Revenue shows a consistent upward trend over time
* Accessories category has the highest order volume
* Return rate remains stable around 2%
* A small group of top products contributes significantly to revenue
* Certain regions consistently outperform others in sales

---

## 💡 Business Recommendations

* Focus marketing efforts on high-performing categories (Accessories)
* Investigate products with relatively higher return rates
* Expand operations in high-performing regions
* Target high-value customers for retention strategies
